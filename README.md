# Market-Vision
Pipeline на Python для еженедельного мониторинга цен конкурентов: сбор с 4 маркетплейсов (Wildberries, Ozon, Золотое Яблоко, Рив Гош)→обработка→4 метода анализа (регрессия, классификация, кластеризация, лингвистика)→Excel-отчёт, дашборд Power BI и веб-панель. Итоговый проект курса «Python. Анализ данных в маркетинге», СПбПУ.
